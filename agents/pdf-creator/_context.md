# pdf-creator

- Role: PDF Studio Pro agent to convert Markdown/HTML into stylized, print-ready PDFs
- Capabilities: Pandoc + LaTeX PDF generation, advanced formatting (tables, code blocks, SVG), cover pages, metadata, optional TOC
- Artifacts: /a0/agents/pdf-creator/ with inputs, styles, outputs
- Prompts: /a0/agents/pdf-creator/prompts containing overrides, reusable via §§include()
