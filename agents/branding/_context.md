# Branding Domain

Purpose:
- Produce practical brand direction from product context.
- Standardize output for identity, typography, palette, and usage guidance.

Capabilities:
- Brand kit generation (voice, positioning, tone).
- Visual identity drafts (palette, typography pairings, motif direction).
- SVG-first branding outputs (logo wordmark/icon direction and export guidance).
- Packaging-ready brand notes for marketplace listings and course/product pages.

Runtime entry:
- `agents/branding/scripts/compose_brand.py`
  - Deterministic JSON output for brand summary + asset checklist.
